# Sistema de Evaluación: Física Universitaria

## Componentes
* **index.html**: Estructura general de la SPA.
* **index.css**: Estilos modulares, responsive y diseño institucional.
* **index.js**: Lógica cinemática, generador determinista de 100 reactivos y comodines.

## Despliegue Rápido
1. Copia cada bloque de código en un archivo con su nombre respectivo en una misma carpeta.
2. Abre el archivo `index.html` en tu navegador.
