# Simulador Científico de Aptitud Académica

Este repositorio contiene 100 ejercicios únicos, indexados matemáticamente y distribuidos rigurosamente en los temas solicitados sin repeticiones de ningún tipo ni temas ajenos.

## Temas Cubiertos
1. Simplificación de expresiones algebraicas
2. Planteamiento de ecuaciones
3. Cálculo de edades
4. Regla de 3 compuesta
5. Porcentajes
6. Media aritmética
7. Razones y proporciones
8. Permutaciones y variaciones
9. Proporcionalidad pura

## Despliegue en GitHub Pages
Para publicar este simulador de forma gratuita:
1. Sube estos 4 archivos a un repositorio público en GitHub.
2. Ve a **Settings** -> **Pages**.
3. En la sección *Build and deployment*, selecciona la rama `main` y la carpeta `/ (root)`.
4. Guarda los cambios. ¡En un par de minutos tu simulador estará en línea listo para compartir!
